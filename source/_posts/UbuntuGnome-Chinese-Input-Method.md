If you like use [UbuntuGnome](http://ubuntugnome.org/), and you encounter the Chinese Input Method problem, here is your solution, first, install the language-support:
```bash
apt-get install language-support
```
second, start the language-spport and install Chinese Simplified pack.

third, start the Input Method configuration and add Chinese Sun-Pinyin input method.

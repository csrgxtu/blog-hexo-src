If you're using [UbuntuGnome](http://ubuntugnome.org/), and want to auto hide the top panel to make full usage of your desktop, then there is a auto hide top panel extension [Panel-Settings](https://github.com/eddiefullmetal/gnome-shell-extensions)
<!--more-->

First, clone the extension to your gnome shell's extension directory.
```bash
git clone https://github.com/eddiefullmetal/gnome-shell-extensions.git
sudo cp -r gnome-shell-extensions/panelSettings@eddiefullmetal.gr /usr/share/gnome-shell/extensions
```

Second, start Tweak tool and select extensions and enable auto-hide.

The reason I write this post is because I am using UbuntuGnome and want to make my desktop a little bigger, and I cant install extensions through Gnome Extensions website directly, so I chose manually install the extensions.
